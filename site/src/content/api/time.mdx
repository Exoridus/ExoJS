# Time
`import { Time } from '@codexo/exojs'`
Time-duration value object stored internally in milliseconds. Provides
unit-converted accessors (`seconds`, `minutes`, `hours`) and arithmetic
helpers (Time.add, Time.subtract, Time.addTime).

Constants on the class hold canonical durations: Time.zero,
Time.oneSecond, Time.oneMinute, Time.oneHour, plus
the TimeInterval multipliers used by the `factor` parameter.

`Time.temp` is a shared scratch instance for hot paths — do not store the
reference across calls.
