## Import

`import { TiledTileLayer } from '@codexo/exojs-tiled'`

A tile layer. On a finite map, data holds the flat row-major array
of GIDs (`width * height` entries). On an infinite map, chunks
holds the sparse list of tile chunks instead; exactly one of the two is
defined, matching the owning TiledMap's `infinite` flag.
