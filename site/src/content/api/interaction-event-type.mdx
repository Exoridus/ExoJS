# InteractionEventType
`import { InteractionEventType } from '@codexo/exojs'`
String literal union of every interaction event the
InteractionManager can deliver to a RenderNode.
Handlers attach via the node's `interactive*` API.
## Definition

`InteractionEventType = "pointerdown" | "pointerup" | "pointermove" | "pointerover" | "pointerout" | "pointertap" | "dragstart" | "drag" | "dragend"`
