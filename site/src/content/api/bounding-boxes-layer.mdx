## Import

`import { BoundingBoxesLayer } from '@codexo/exojs/debug'`

Debug layer that draws a bounding-box outline around every visible
RenderNode in the scene. Colours cycle by zIndex (HSL hue). World-space.

## Constructors

- `new(app: Application): BoundingBoxesLayer`

## Methods

- `destroy(): void` - Release the internal Graphics primitive.
- `render(backend: RenderBackend): void` - Walk the scene tree and draw a colored rectangle outline around every visible node that has non-zero bounds. Outline hue cycles by zIndex so overlapping nodes are visually distinguishable.
- `update(_delta: Time): void` - No per-frame pre-computation required; all state is derived in render.

## Properties

- `visible: boolean`
- `viewMode: DebugLayerViewMode` - Renders in world-space so outlines align with scene node positions.
