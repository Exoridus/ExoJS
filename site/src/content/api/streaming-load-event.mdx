# StreamingLoadEvent
`import { StreamingLoadEvent } from '@codexo/exojs'`
`HTMLMediaElement` ready-state events the streaming asset factories
(MusicFactory, VideoFactory) accept as the load-completion
signal. Pick the earliest event that meets your latency requirement;
`'canplaythrough'` is the safest default.
## Definition

`StreamingLoadEvent = "loadedmetadata" | "loadeddata" | "canplay" | "canplaythrough"`
