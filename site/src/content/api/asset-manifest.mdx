# AssetManifest
`import { AssetManifest } from '@codexo/exojs'`
Static description of all asset bundles in an application.

Pass to Loader.registerManifest and then load individual bundles on
demand with Loader.loadBundle. Use defineAssetManifest to
construct a validated, type-safe manifest at authoring time.
## Properties

- `bundles: Readonly<Record<string, readonly AssetEntry[]>>`
