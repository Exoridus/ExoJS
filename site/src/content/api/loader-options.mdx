# LoaderOptions
`import { LoaderOptions } from '@codexo/exojs'`
Construction options for Loader.

`resourcePath` is prepended to relative asset paths at fetch time.
`cache` accepts one or more CacheStore instances. `cacheStrategy`
picks the policy used to consult them — defaults to
CacheFirstStrategy (check stores → network → write back).
`concurrency` caps the number of simultaneous background-queue fetches
(default `6`).
## Properties

- `cache?: CacheStore | readonly CacheStore[]`
- `cacheStrategy?: CacheStrategy`
- `concurrency?: number`
- `requestOptions?: RequestInit`
- `resourcePath?: string`
