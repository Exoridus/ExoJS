# CacheStore
`import { CacheStore } from '@codexo/exojs'`
Persistent storage backend used by the Loader to cache processed
asset data between sessions.

Each method accepts a `storageName` namespace (sourced from
AssetFactory.storageName) and a per-asset `key`, allowing a single
store to hold data for multiple asset types. IndexedDbStore is the
built-in implementation backed by the browser's IndexedDB API.
