# CollisionType
`import { CollisionType } from '@codexo/exojs'`
Discriminant tag carried by every Collidable so that intersection
and collision routines can dispatch to the correct algorithm without
`instanceof` checks.
## Members

- `Circle`
- `Ellipse`
- `Line`
- `Point`
- `Polygon`
- `Rectangle`
- `SceneNode`
