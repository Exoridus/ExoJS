## Import

`import { Logger } from '@codexo/exojs'`

Engine-wide log dispatcher. Below `LogSeverity.Error`, calls are dropped
outright in production builds (`__DEV__` is `false`); `error()` always
reaches every registered sink. In development a createConsoleSink
is installed by default — see logger.
