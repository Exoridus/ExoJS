## Import

`import { FlangerEffect } from '@codexo/exojs-audio-fx'`

Flanger effect using native WebAudio nodes (no worklet required).

A flanger is a very short LFO-modulated delay (1вҖ“10 ms) with a feedback
loop. Unlike a chorus вҖ” which uses longer delays without feedback to produce
thickening вҖ” the flanger's feedback reinforces phase cancellation, creating
the characteristic sweeping comb-filter "jet-plane" sound as notches and
peaks slide up and down the spectrum.

                 вҶ‘    в”Ӯ
              feedbackGain в—Җв”Җв”ҳ

lfoOscillator в”Җв”Җ lfoGain (depth in seconds) в”Җв”Җв–¶ delayNode.delayTime

Gains: `dryGain.gain = 1 - wet`; `wetGain.gain = wet`;
`feedbackGain.gain = feedback`; `lfoGain.gain = depthMs / 1000`.
