## Import

`import { XboxGamepadMapping } from '@codexo/exojs'`

Mapping for Microsoft Xbox controllers (Xbox One, Xbox Series X/S, and
compatible third-party XInput devices) connected via USB or Bluetooth.

Inherits the full GenericDualAnalogGamepadMapping layout, which
mirrors the W3C standard gamepad layout that XInput controllers follow
natively. The Share button (Xbox Series) maps to the Share channel.

## Constructors

- `new(): XboxGamepadMapping`

## Methods

- `destroy(): void` - Releases all button and axis control references held by this mapping. Call when the associated gamepad disconnects to allow garbage collection.
- `hasChannel(channel: GamepadButtonChannel | GamepadAxisChannel): boolean` - Returns true when this mapping declares at least one button or axis control that writes to channel. Use to detect device-specific capabilities at runtime — e.g. before binding an input to a right-stick channel that may not exist on a single Joy-Con.

## Properties

- `axes: readonly GamepadAxis[]` - Ordered list of axes, indexed by the Gamepad API axis index.
- `buttons: readonly GamepadButton[]` - Ordered list of buttons, indexed by the Gamepad API button index.
- `family: Xbox` - Identifies the device family this mapping targets.
