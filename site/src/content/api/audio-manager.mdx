## Import

`import { AudioManager } from '@codexo/exojs'`

Module-level singleton that owns the engine-wide audio mix: three
pre-configured AudioBus instances (`master` ← `music` + `sound`),
a single shared AudioListener for spatial audio, and a registry
of any extra busses the user constructs.

Access the singleton via getAudioManager (or `app.audio`). The
instance is constructed lazily on first access so client code that does
not need audio pays no startup cost.

Drives the per-frame `_tick()` on the listener and every spatial sound;
propagates visibility-driven mute when AudioManager.muteOnHidden
is enabled.
