## Import

`import { PerformanceLayer } from '@codexo/exojs/debug'`

Debug layer that renders a compact screen-space HUD (top-left) showing
rolling-average FPS, per-frame time in milliseconds, draw-call count, and
scene-node count, alongside a 120-sample frame-time sparkline.

Enable via DebugOverlay or by pressing F1 while the canvas has focus.

## Constructors

- `new(app: Application): PerformanceLayer`

## Methods

- `destroy(): void` - Destroy the panel's Container subtree and release all child references.
- `render(backend: RenderBackend): void` - Submit the panel's Container subtree to the backend for drawing.
- `update(delta: Time): void` - Sample the current frame time, recompute the rolling FPS average, update stat text nodes, and rebuild the sparkline geometry. Lazily initializes the panel scene graph on first call to avoid touching the glyph atlas until the layer is actually shown.

## Properties

- `visible: boolean`
- `viewMode: DebugLayerViewMode` - Whether this layer renders in the scene's world-space view or in a screen-space (canvas-pixel) view. Defaults to 'screen'. Subclasses override to return 'world' when rendering at actual scene positions.
