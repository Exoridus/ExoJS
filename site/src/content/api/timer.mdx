## Import

`import { Timer } from '@codexo/exojs'`

Clock variant with a fixed limit. Inherits start/stop/reset/restart
semantics; adds Timer.expired (true once `elapsedTime >= limit`)
and remaining-time accessors. Useful for cooldowns, delays, and any timed
gating logic where you want to ask "is the duration up?" each frame.
