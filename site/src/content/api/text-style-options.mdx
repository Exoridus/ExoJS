# TextStyleOptions
`import { TextStyleOptions } from '@codexo/exojs'`
Construction-time options for a TextStyle.
All properties are optional; defaults match the TextStyle constructor defaults.
## Properties

- `align?: TextAlignment`
- `baseline?: CanvasTextBaseline`
- `fill?: TextStyleColor`
- `fillColor?: Color<>`
- `fontFamily?: string`
- `fontSize?: number`
- `fontStyle?: "normal" | "italic"`
- `fontWeight?: string`
- `lineHeight?: number`
- `lineJoin?: CanvasLineJoin`
- `miterLimit?: number`
- `padding?: number`
- `stroke?: TextStyleColor`
- `strokeThickness?: number`
- `wordWrap?: boolean`
- `wordWrapWidth?: number`
