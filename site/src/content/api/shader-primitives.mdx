# ShaderPrimitives
`import { ShaderPrimitives } from '@codexo/exojs'`
GLSL primitive type tokens used to describe ShaderAttribute and ShaderUniform data types.
Values are WebGL2 GLenum constants returned by `gl.getActiveAttrib` / `gl.getActiveUniform`.
## Members

- `Bool`
- `BoolVec2`
- `BoolVec3`
- `BoolVec4`
- `Float`

- `FloatVec2`
- `FloatVec3`
- `FloatVec4`
- `Int`

- `IntVec3`
- `IntVec4`
- `Sampler2D`
- `UnsignedInt`
- `UnsignedIntVec2`
- `UnsignedIntVec3`
- `UnsignedIntVec4`
