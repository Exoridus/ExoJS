## Import

`import { ShaderAttribute } from '@codexo/exojs'`

Metadata for a single vertex attribute declared in a GLSL vertex shader.

Populated by the backend during Shader.connect using the values
returned by `gl.getActiveAttrib`. The `size` field is derived from `type`
via the backend's primitive-byte-size mapping and is used to stride vertex
buffer layouts.

## Constructors

- `new(index: number, name: string, type: number): ShaderAttribute`

## Methods

- `destroy(): void`

## Properties

- `index: number` - Zero-based index of this attribute within the shader program.
- `location: number` - WebGL attribute location assigned by the driver. -1 until the backend sets it.
- `name: string` - GLSL attribute name as declared in the vertex shader source.
- `size: number` - Byte size of one element of this attribute's type.
- `type: number` - WebGL2 GLenum type token (e.g. gl.FLOAT_VEC2).
