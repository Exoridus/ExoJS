## Import

`import { TiledObject } from '@codexo/exojs-tiled'`

A parsed Tiled object, placed in an TiledObjectLayer or a tile's
collision `objectgroup`.

The shape of the object is determined by which of point,
ellipse, polygon, polyline, text, or
gid is set; if none are set, the object is a plain rectangle
spanning `[x, y, width, height]`.
