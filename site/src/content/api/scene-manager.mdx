## Import

`import { SceneManager } from '@codexo/exojs'`

Stack-based scene controller owned by Application. Maintains an
ordered stack of Scene instances, each tagged with its
participation policy (SceneStackMode).
Scenes higher on the stack overlay scenes lower; the policy of each
scene determines whether scenes below continue to update / render.

Use SceneManager.setScene to replace the entire stack with one
scene, SceneManager.pushScene to overlay a new scene on top, and
SceneManager.popScene to remove the topmost. All three accept an
optional fade transition.
