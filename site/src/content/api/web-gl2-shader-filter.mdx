## Import

`import { WebGl2ShaderFilter } from '@codexo/exojs'`

A high-level Filter subclass that renders the input texture
through a user-provided GLSL fragment shader on the **WebGL2** backend.

For the WebGPU backend use WebGpuShaderFilter.

## Usage

```ts
const filter = new WebGl2ShaderFilter(\{
  fragmentSource: `
    #version 300 es
    precision mediump float;
    uniform sampler2D uTexture;
    uniform vec2 uResolution;
    uniform float uTime;
    in vec2 vUv;
    out vec4 fragColor;
    void main() \{
      fragColor = texture(uTexture, vUv);
    \}
  `,
  uniforms: \{ uTime: 0.0 \},
\});

// Update uniforms each frame:
filter.uniforms.uTime = performance.now() / 1000;
sprite.filters = [filter];
```

## Auto-bound uniforms

The backend automatically sets `uTexture` (slot 0) and `uResolution`
before each draw. User uniforms start at texture slot 1.

## Constructors

- `new(options: WebGl2ShaderFilterOptions): WebGl2ShaderFilter`

## Methods

- `apply(backend: RenderBackend, input: RenderTexture, output: RenderTexture): void` - Execute the GLSL shader pass: compile the program on first call, bind uniforms, and render the input texture into output. Throws if the active backend is WebGPU — use WebGpuShaderFilter on WebGPU.
- `destroy(): void` - Release any GPU-side resources held by this filter (uniform buffers, pipelines, intermediate textures). Default is a no-op for stateless filters; subclasses with state (BlurFilter, ColorFilter) override.

## Properties

- `uniforms: Record<string, ShaderFilterUniformValue>` - Mutable map of uniform values. Set values via property assignment; they are flushed to the GPU before each apply(). filter.uniforms.uTime = 1.234; filter.uniforms.uColor = [1, 0.5, 0, 1]; // vec4
