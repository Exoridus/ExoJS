# BundleLoadError
`import { BundleLoadError } from '@codexo/exojs'`
Thrown by Loader.loadBundle when one or more assets in the bundle
fail to load.

The `failures` array contains every entry that errored, letting callers
distinguish individual per-asset failures from a wholesale network outage.

- `new(bundle: string, failures: object[]): BundleLoadError<>`
