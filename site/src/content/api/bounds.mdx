## Import

`import { Bounds } from '@codexo/exojs'`

Mutable axis-aligned bounding box accumulator. Starts as the empty bounds
(`-Infinity..Infinity` reversed) and grows monotonically as points or
rectangles are merged in via Bounds.addCoords and
Bounds.addRect. Used during scene-graph traversal to accumulate a
subtree's combined extent for culling and hit-testing.

The Bounds.getRect accessor lazily folds the current min/max into
a shared Rectangle — copy the result if you need to retain it
across further accumulation.

## Constructors

- `new(): Bounds`

## Methods

- `addCoords(x: number, y: number): this` - Expand the bounds to include (x, y).
- `addRect(rectangle: Rectangle, transform?: Matrix): this` - Expand the bounds to include rectangle. When transform is provided the rectangle's corners are transformed into the bounds' coordinate space first (via Rectangle.temp — does not mutate rectangle).
- `destroy(): void`
- `getRect(): Rectangle` - Materialize the accumulated min/max as a Rectangle. Returns the same instance across calls — copy if you need to retain it past the next mutation.
- `reset(): this` - Restore the empty-bounds state for reuse across frames.
