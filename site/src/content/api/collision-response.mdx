# CollisionResponse
`import { CollisionResponse } from '@codexo/exojs'`
Result of a successful Collidable.collidesWith call. Contains
the two participating shapes, the penetration depth, containment flags,
and the minimum-translation vectors (normal + vector) needed to separate
them.
## Properties

- `overlap: number`
- `projectionN: Vector<>`
- `projectionV: Vector<>`
- `shapeA: Collidable`
- `shapeAinB: boolean`
- `shapeB: Collidable`
- `shapeBinA: boolean`
