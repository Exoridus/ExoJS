## Import

`import { AbstractText } from '@codexo/exojs'`

Base class for all text rendering nodes. Provides the common `text`
property and the on-demand dirty protocol used by the renderer.

Subclasses:
- Text — runtime Canvas 2D / SDF rasterization
- BitmapText  — offline pre-built atlas (BMFont / MSDF)
