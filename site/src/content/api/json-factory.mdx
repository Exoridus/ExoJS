## Import

`import { JsonFactory } from '@codexo/exojs'`

AssetFactory implementation that parses JSON files and exposes them
as plain JavaScript values.

The parsed value is typed as `unknown`; callers are expected to narrow or
cast the result to their specific schema type.

## Constructors

- `new(): JsonFactory`

## Methods

- `create(source: unknown): Promise<unknown>` - Returns the already-parsed value unchanged.
- `createObjectUrl(blob: Blob): string` - Creates an object URL from blob and registers it for automatic cleanup when destroy is called. Prefer this over calling URL.createObjectURL directly inside a factory subclass so the URL is always tracked.
- `destroy(): void` - Revokes every object URL that was created via createObjectUrl and clears the internal tracking pool.
- `process(response: Response): Promise<unknown>` - Parses the response body as JSON and returns the resulting value.
- `revokeObjectUrl(objectUrl: string): void`

## Properties

- `storageName: "json"` - Identifier used as the object-store / storage-namespace key when this factory's assets are persisted to a CacheStore.
