# LoadReturn
`import { LoadReturn } from '@codexo/exojs'`
Maps a Loadable constructor to the concrete type returned by
Loader.load.

Token classes (Json, TextAsset, SvgAsset,
VttAsset) are special-cased because their return types do not match
their constructor type. All other loadables return the instance type inferred
from the constructor.
## Definition

`LoadReturn = T extends typeof Json ? unknown : T extends typeof TextAsset ? string : T extends typeof SvgAsset ? HTMLImageElement : T extends typeof VttAsset ? VTTCue[] : T extends object ? R : never`
