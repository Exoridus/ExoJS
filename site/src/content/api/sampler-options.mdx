# SamplerOptions
`import { SamplerOptions } from '@codexo/exojs'`
Full set of texture sampling and upload parameters.
Used by both Texture and RenderTexture to configure the GPU sampler
and the pixel-store state at upload time.
## Properties

- `flipY: boolean`
- `generateMipMap: boolean`
- `premultiplyAlpha: boolean`
- `scaleMode: ScaleModes`
- `wrapMode: WrapModes`
