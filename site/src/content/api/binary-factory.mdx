## Import

`import { BinaryFactory } from '@codexo/exojs'`

AssetFactory implementation that loads arbitrary binary files and
exposes them as raw ArrayBuffer instances.

Use this for any binary format that does not have a dedicated factory (e.g.
custom data files, packed archives, or proprietary formats).

## Constructors

- `new(): BinaryFactory`

## Methods

- `create(source: ArrayBuffer): Promise<ArrayBuffer>` - Returns the raw buffer unchanged — no further transformation is applied.
- `createObjectUrl(blob: Blob): string` - Creates an object URL from blob and registers it for automatic cleanup when destroy is called. Prefer this over calling URL.createObjectURL directly inside a factory subclass so the URL is always tracked.
- `destroy(): void` - Revokes every object URL that was created via createObjectUrl and clears the internal tracking pool.
- `process(response: Response): Promise<ArrayBuffer>` - Reads the full response body as an ArrayBuffer.
- `revokeObjectUrl(objectUrl: string): void`

## Properties

- `storageName: "binary"` - Identifier used as the object-store / storage-namespace key when this factory's assets are persisted to a CacheStore.
