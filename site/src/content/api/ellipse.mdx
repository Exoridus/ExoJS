# Ellipse
`import { Ellipse } from '@codexo/exojs'`
Mutable axis-aligned ellipse shape defined by a centre position and
separate horizontal (`rx`) and vertical (`ry`) half-radii. Implements
ShapeLike for use in the intersection pipeline.

`collidesWith` returns a CollisionResponse for ellipse-vs-rectangle
(axis-aligned), ellipse-vs-circle, and ellipse-vs-SceneNode (which
collides against the node's AABB). Other targets — ellipse-vs-ellipse,
ellipse-vs-polygon, and ellipse-vs-line — return `null`; use
Ellipse.intersectsWith for boolean tests against those.
