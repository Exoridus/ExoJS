# TweenState
`import { TweenState } from '@codexo/exojs'`
Lifecycle states of a Tween. A tween starts in `Idle`, transitions
to `Active` once Tween.start is called, and ends in `Complete`
(natural finish, all repeats exhausted) or `Stopped` (cancelled via
Tween.stop). `Paused` is reachable from `Active` only.
## Members

- `Active`
- `Complete`
- `Idle`
- `Paused`
- `Stopped`
