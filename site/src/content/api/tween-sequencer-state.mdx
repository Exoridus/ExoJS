## Import

`import { TweenSequencerState } from '@codexo/exojs'`

Lifecycle states of a TweenSequencer. Mirrors TweenState
semantics: starts `Idle`, becomes `Active` on TweenSequencer.start,
and ends in `Complete` (all stages exhausted) or `Stopped` (cancelled via
TweenSequencer.stop). `Paused` is reachable from `Active` only.

## Members

- `Active`
- `Complete`
- `Idle`
- `Paused`
- `Stopped`
