## Import

`import { PointerStateFlag } from '@codexo/exojs'`

Bit flags accumulated on a Pointer between frames so consumers can
detect transient events (entered the canvas, was released, was cancelled)
even if the pointer's `currentState` has already moved on. Cleared each
frame by the InteractionManager.

## Members

- `Cancel`
- `Down`
- `Leave`
- `Move`
- `None`
- `Over`
- `Up`
