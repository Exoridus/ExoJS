# ShapeLike
`import { ShapeLike } from '@codexo/exojs'`
Full shape contract: a Collidable that can also be cloned, destroyed,
and queried for its axis-aligned bounding box. All concrete shape classes
(Circle, Rectangle, Polygon, Ellipse,
Line, Vector) implement this interface.
