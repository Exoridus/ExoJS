## Import

`import { AudioFilter } from '@codexo/exojs'`

Abstract base for all audio processing filters in ExoJS. Each filter exposes
stable inputNode and outputNode AudioNodes that the audio bus
connects into the Web Audio graph. Async filters (e.g., worklet-based) expose
a ready promise that resolves once the filter is fully initialized.
