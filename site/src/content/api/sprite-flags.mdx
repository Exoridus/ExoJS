# SpriteFlags
`import { SpriteFlags } from '@codexo/exojs'`
Internal dirty-flag bitmask used by Sprite to lazily recompute
derived data (vertices, normals, texture coordinates, bounding boxes).

- `None`
- `Normals`
- `Origin`
- `Rotation`
- `Scaling`
- `TextureCoords`
- `Transform`
- `TransformInverse`
- `Translation`
- `VertexTint`
- `Vertices`
