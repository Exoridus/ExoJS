# TextureSource
`import { TextureSource } from '@codexo/exojs'`
Anything the rendering pipeline can sample as a texture source: a loaded
image, an offscreen canvas, a playing video, or `null` when no source has
been assigned yet.
## Definition

`TextureSource = HTMLImageElement | HTMLCanvasElement | HTMLVideoElement | null`
