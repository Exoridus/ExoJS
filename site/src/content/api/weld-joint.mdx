## Import

`import { WeldJoint } from '@codexo/exojs-physics'`

Rigidly fixes the relative position and orientation of two bodies (they move
as one rigid body). A 2-DOF point constraint (like RevoluteJoint) plus
a 1-DOF angular constraint, solved in the sub-step loop. Both locks default to
rigid; set `linearHertz`/`angularHertz` for a springy weld.

## Constructors

- `new(options: WeldJointOptions): WeldJoint`

## Properties

- `angularHertz: number` - Soft frequency for the angle lock (0 = rigid).
- `bodyA: PhysicsBody` - First constrained body.
- `bodyB: PhysicsBody` - Second constrained body.
- `dampingRatio: number` - Soft damping ratio.
- `enabled: boolean` - When false, the joint is skipped by the solver (but still tracked by the world).
- `linearHertz: number` - Soft frequency for the position lock (0 = rigid).
- `referenceAngle: number` - Locked relative angle angleB − angleA.
