# DebugLayerViewMode
`import { DebugLayerViewMode } from '@codexo/exojs/debug'`
Determines the coordinate space a DebugLayer renders into.
`'screen'` uses the overlay's pixel-space view; `'world'` uses the scene's
current world-space view and renders beneath screen-space panels.
## Definition

`DebugLayerViewMode = "screen" | "world"`
