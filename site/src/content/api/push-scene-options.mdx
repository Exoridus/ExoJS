# PushSceneOptions
`import { PushSceneOptions } from '@codexo/exojs'`
Options passed to SceneManager.pushScene. Inherits
SceneParticipationPolicy so the pushed scene's stack/input mode
can be overridden at the call site without subclassing.
## Properties

- `input?: SceneInputMode`
- `mode?: SceneStackMode`
- `transition?: FadeSceneTransition`
