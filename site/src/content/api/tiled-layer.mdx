## Import

`import { TiledLayer } from '@codexo/exojs-tiled'`

Base class for the four parsed Tiled layer types. Holds the fields shared
by every layer (`tilelayer`, `objectgroup`, `imagelayer`, `group`).

Use TiledLayer.type (or `instanceof`) to discriminate between
TiledTileLayer, TiledObjectLayer, TiledImageLayer,
and TiledGroupLayer.
