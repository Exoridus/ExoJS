## Import

`import { TextureFactory } from '@codexo/exojs'`

AssetFactory implementation that loads PNG, JPG, WebP, AVIF, and
other browser-supported raster image formats and produces a GPU-ready
Texture instance.

MIME type detection is performed automatically from the buffer's magic bytes;
pass an explicit `mimeType` to override. Sampler state (wrap, filter, etc.)
can be configured via `samplerOptions`.

- `new(): TextureFactory`

## Methods

- `create(source: ArrayBuffer, options: TextureFactoryOptions): Promise<Texture>` - Decodes image bytes and wraps the result in a Texture. Prefers createImageBitmap for a zero-copy GPU-upload path. Falls back to a temporary object URL + \<img> element on environments that do not support createImageBitmap.
- `createObjectUrl(blob: Blob): string` - Creates an object URL from blob and registers it for automatic cleanup when destroy is called. Prefer this over calling URL.createObjectURL directly inside a factory subclass so the URL is always tracked.
- `destroy(): void` - Revokes every object URL that was created via createObjectUrl and clears the internal tracking pool.
- `process(response: Response): Promise<ArrayBuffer>` - Reads the full response body as an ArrayBuffer for MIME-type detection and blob construction.
- `revokeObjectUrl(objectUrl: string): void`

## Properties

- `storageName: "texture"` - Identifier used as the object-store / storage-namespace key when this factory's assets are persisted to a CacheStore.
