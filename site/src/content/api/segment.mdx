## Import

`import { Segment } from '@codexo/exojs'`

Mutable line segment defined by start and end Vectors. Unlike
Line this type does not implement ShapeLike — it is a
lightweight value type used in path and swept-collision utilities.

`Segment.temp` is a shared scratch instance.
