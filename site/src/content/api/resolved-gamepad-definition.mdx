# ResolvedGamepadDefinition
`import { ResolvedGamepadDefinition } from '@codexo/exojs'`
The fully resolved product of running a GamepadDefinition against a
connected gamepad — bundles the original descriptor, the resolved display name,
and the chosen GamepadMapping together for use by Gamepad.

- `descriptor: GamepadDescriptor`
- `mapping: GamepadMapping<>`
- `name: string`
