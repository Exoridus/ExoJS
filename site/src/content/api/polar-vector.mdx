## Import

`import { PolarVector } from '@codexo/exojs'`

A 2D vector expressed in polar coordinates: `radius` (magnitude) and `phi`
(angle in radians, measured from the positive X-axis).

## Constructors

- `new(radius: number, angle: number): PolarVector`

## Methods

- `toVector(): Vector` - Convert to a Cartesian Vector. Returns Vector.temp — do not store the reference across calls.
- `fromVector(vector: Vector): PolarVector` - Create a PolarVector from a Cartesian vector, preserving both magnitude and direction. The roundtrip PolarVector.fromVector(v).toVector() reproduces v (modulo float precision).

## Properties

- `phi: number` - Angle in radians, measured from the positive X-axis.
- `radius: number`
