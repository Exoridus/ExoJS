# BufferUsage
`import { BufferUsage } from '@codexo/exojs'`
Hints describing expected buffer access pattern, allowing the driver to optimise allocation.
Values are WebGL2 GLenum constants used when calling `gl.bufferData`.
## Members

- `DynamicCopy`
- `DynamicDraw`
- `DynamicRead`
- `StaticCopy`
- `StaticDraw`
- `StaticRead`
- `StreamCopy`
- `StreamDraw`
- `StreamRead`
