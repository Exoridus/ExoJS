# AudioSpriteClip
`import { AudioSpriteClip } from '@codexo/exojs'`
Named sub-region of an AudioBuffer used as an audio sprite sheet.
`start` / `end` are seconds into the buffer; `loop` makes
Sound.playSprite loop the sprite indefinitely.
## Properties

- `end: number`
- `loop?: boolean`
- `start: number`
