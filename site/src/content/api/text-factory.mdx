## Import

`import { TextFactory } from '@codexo/exojs'`

AssetFactory implementation that loads plain-text files (TXT, CSV,
GLSL shaders, HTML fragments, and any other UTF-8 text) and exposes them as
JavaScript strings.

## Constructors

- `new(): TextFactory`

## Methods

- `create(source: string): Promise<string>` - Returns the decoded string unchanged.
- `createObjectUrl(blob: Blob): string` - Creates an object URL from blob and registers it for automatic cleanup when destroy is called. Prefer this over calling URL.createObjectURL directly inside a factory subclass so the URL is always tracked.
- `destroy(): void` - Revokes every object URL that was created via createObjectUrl and clears the internal tracking pool.
- `process(response: Response): Promise<string>` - Reads the response body as a UTF-8 decoded string.
- `revokeObjectUrl(objectUrl: string): void`

## Properties

- `storageName: "text"` - Identifier used as the object-store / storage-namespace key when this factory's assets are persisted to a CacheStore.
