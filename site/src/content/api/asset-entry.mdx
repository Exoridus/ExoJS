# AssetEntry
`import { AssetEntry } from '@codexo/exojs'`
A single asset declaration inside an AssetManifest bundle.

`type` is the loadable class token (e.g. `Texture`, `Sound`), `alias` is
the key used to retrieve the asset from the Loader, and `path` is
the URL or relative path used to fetch it.
## Properties

- `alias: string`
- `options?: unknown`
- `path: string`
- `type: T`
