# GamepadMappingFamily
`import { GamepadMappingFamily } from '@codexo/exojs'`
Discriminant tag identifying which device family a GamepadMapping belongs to.
Used to select the correct mapping at runtime when a gamepad connects.

- `GameCube`
- `GenericDualAnalog`
- `JoyConLeft`
- `JoyConRight`
- `PlayStation`
- `SteamController`
- `SteamDeck`
- `SwitchPro`
- `Xbox`
