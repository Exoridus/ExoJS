## Import

`import { PointerStackLayer } from '@codexo/exojs/debug'`

Debug layer that shows a screen-space panel (top-right corner) listing all
RenderNodes under the current pointer position, sorted by zIndex descending.
Screen-space.

## Constructors

- `new(app: Application): PointerStackLayer`

## Methods

- `destroy(): void` - Destroy the panel's Container subtree and release all child references.
- `render(backend: RenderBackend): void` - Submit the panel's Container subtree to the backend for drawing.
- `update(_delta: Time): void` - Lazily initialize the panel on first call, reposition it to the top-right corner, and populate text lines with the current pointer stack (up to 10 entries, sorted by zIndex descending).

## Properties

- `visible: boolean`
- `viewMode: DebugLayerViewMode` - Renders in screen-space so the panel stays fixed in the top-right corner.
