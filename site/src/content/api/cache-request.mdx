# CacheRequest
`import { CacheRequest } from '@codexo/exojs'`
All the information a CacheStrategy needs to resolve a single asset.

Bundles the target factory, cache namespace, lookup key, the fetch
parameters, and any factory-specific options into one value so strategies
remain stateless.
## Properties

- `factory: AssetFactory<>`
- `key: string`
- `options?: unknown`
- `requestOptions: RequestInit`
- `storageName: string`
- `url: string`
