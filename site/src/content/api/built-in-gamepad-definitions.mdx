# builtInGamepadDefinitions
`import { builtInGamepadDefinitions } from '@codexo/exojs'`
The default ordered list of GamepadDefinition entries used by
resolveGamepadDefinition when no custom list is supplied.

Ordered as: exact product-ID matches → vendor-ID fallbacks → generic catch-all.
Register custom definitions by prepending to a copy and passing it explicitly.
## Type

`builtInGamepadDefinitions: GamepadDefinition[]`
