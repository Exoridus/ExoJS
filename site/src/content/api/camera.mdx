## Import

`import { Camera } from '@codexo/exojs'`

User-facing camera abstraction. Extends View with ergonomic
construction defaults. Use as the default camera owned by
RenderingContext or pass custom instances to
`context.render(node, \{ view: myCamera \})`.

```ts
const cam = new Camera(\{ center: \{ x: 50, y: 50 \}, size: \{ width: 800, height: 600 \} \});
```

Defaults: center `(0, 0)`, size `\{ width: 0, height: 0 \}`, full viewport
`(0, 0, 1, 1)`, rotation `0`, zoom `1`.
