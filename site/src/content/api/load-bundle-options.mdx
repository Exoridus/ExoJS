# LoadBundleOptions
`import { LoadBundleOptions } from '@codexo/exojs'`
Options controlling how a bundle is loaded by Loader.loadBundle.

Set `background` to `true` to load the bundle through the low-priority
background queue, and supply `onProgress` for per-bundle progress updates.
## Properties

- `background?: boolean`
- `onProgress?: object`
