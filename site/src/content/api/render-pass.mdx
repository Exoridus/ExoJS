# RenderPass
`import { RenderPass } from '@codexo/exojs'`
A single node in the render graph.

Concrete implementations include CallbackRenderPass for arbitrary
draw logic and RenderTargetPass for off-screen rendering to a
RenderTarget. Passes are executed in order each frame by the backend.
