## Import

`import { HitTestLayer } from '@codexo/exojs/debug'`

Debug layer that draws outlines around interactive scene nodes.
- Magenta: interactive but idle.
- Yellow: currently hovered.
- Cyan: pointer-captured (drag in progress).
- Faint gray quadrant outlines when the spatial index is active (i.e.
  at least one interactive node is present in the scene).
World-space.

## Constructors

- `new(app: Application): HitTestLayer`

## Methods

- `destroy(): void` - Release the internal Graphics primitive.
- `render(backend: RenderBackend): void` - Walk the scene tree and draw color-coded outlines around interactive nodes (magenta = idle, yellow = hovered, cyan = pointer-captured). Also draws faint gray quadrant boundaries when the spatial index is active, helping diagnose partitioning behavior.
- `update(_delta: Time): void` - No per-frame pre-computation required; all state is derived in render.

## Properties

- `visible: boolean`
- `viewMode: DebugLayerViewMode` - Renders in world-space so outlines align with interactive node positions.
