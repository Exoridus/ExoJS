# IndexedDbStoreOptions
`import { IndexedDbStoreOptions } from '@codexo/exojs'`
Configuration options for IndexedDbStore.

`name` is the IndexedDB database name; `version` defaults to `1`.
`storeNames` overrides the default asset-type object stores, and
`migrations` provides explicit per-version schema callbacks (see
IndexedDbDatabase for migration semantics).
## Properties

- `migrations?: Record<number, object>`
- `name: string`
- `storeNames?: readonly string[]`
- `version?: number`
