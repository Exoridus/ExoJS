## Import

`import { ShaderUniform } from '@codexo/exojs'`

Metadata and current value for a single GLSL uniform variable.

Populated by the backend during Shader.connect. The `dirty` flag is
set to `true` on construction and on every setValue call; the backend
clears it via markClean after uploading the value to the GPU.
Array-type uniform names have their index suffix stripped (e.g. `uColors[0]`
becomes `uColors`).

## Constructors

- `new(index: number, type: number, size: number, name: string, data: TypedArray): ShaderUniform`

## Methods

- `destroy(): void`
- `setValue(value: TypedArray): this` - Copy value into the internal typed-array buffer and mark the uniform dirty. The backend will upload the new value to the GPU on the next Shader.sync call.

## Properties

- `index: number` - Zero-based index of this uniform within the shader program.
- `name: string` - GLSL uniform name with array suffixes stripped.
- `size: number` - Number of elements for array uniforms; 1 for scalar uniforms.
- `type: number` - WebGL2 GLenum type token (e.g. gl.FLOAT_MAT4).
- `dirty: boolean` - Whether the value has changed since the last markClean call.
- `propName: string` - The leaf segment of a dot-separated uniform name. For top-level uniforms this equals name; for struct members (e.g. uLight.color) it returns only the field name (color).
- `value: TypedArray`
