## Import

`import { GamepadPromptLayouts } from '@codexo/exojs'`

Static utility that drives in-game controller-prompt UI.

Provides the canonical set of prompt controls, their normalised [x, y] positions
on a generic controller silhouette, device-family label strings (e.g. "A" for
Xbox, "Cross" for PlayStation, "B" for Switch), and the mapping from prompt
control names to GamepadButton channel values.

## Constructors

- `new(): GamepadPromptLayouts`

## Methods

- `getControlChannelMap(): ReadonlyMap<GamepadPromptControl, GamepadButtonChannel>` - Returns the static mapping from each GamepadPromptControl to its corresponding GamepadButton channel. The composite 'DPad' control has no channel entry and is absent from the returned map.
- `getControlLabels(family: GamepadMappingFamily): ReadonlyMap<GamepadPromptControl, string>` - Returns the label map for the given device family, e.g. \{ ButtonSouth → "A" \} for Xbox or \{ ButtonSouth → "Cross" \} for PlayStation. Falls back to generic labels when family has no registered label set.
- `getControlPosition(control: GamepadPromptControl): readonly [number, number]` - Returns the normalised [x, y] position of control on a generic controller silhouette, where (0, 0) is the top-left and (1, 1) the bottom-right. Falls back to [0.5, 0.5] (centre) when the control has no registered position.

## Properties

- `controls: GamepadPromptControl[]` - Complete ordered list of every GamepadPromptControl token.
